# Resume — Vinay Pokharkar

LaTeX source for my resume. The built PDF lives at [`docs/Vinay_Pokharkar_Resume.pdf`](docs/Vinay_Pokharkar_Resume.pdf) and is regenerated automatically on every push to `main`.

## Build locally

```bash
pdflatex resume.tex
```
